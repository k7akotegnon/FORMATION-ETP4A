# Mini jeu POO - Version 1.0

Petit jeu de combat en Ruby pour pratiquer la programmation orientée objet.

## Principe

Deux joueurs s'affrontent :
- chacun commence avec 10 points de vie
- chaque attaque inflige un nombre aléatoire de dégâts (1 à 6)
- le combat continue jusqu'à la mort d'un joueur

## Lancer le jeu

```bash
ruby app.rb
