Bienvenue dans ce projet d'initiation à la Programmation Orientée Objet (POO) en Ruby. L'objectif est de manipuler des classes, des variables d'instance et des méthodes pour gérer des utilisateurs et des événements.

## 📝 Résumé
Le programme permet de créer et gérer deux types d'objets principaux :  
**User** : Pour gérer les utilisateurs du système.  
**Event** : Pour créer des événements avec un titre, une date de début, une durée et une liste de participants.   
Le programme permet d'afficher les détails de l'événement de manière stylée dans le terminal. 

## 📂 Structure du projet
Le projet suit une architecture Ruby classique :  
mon_projet  
├── lib  
│   ├── event.rb       # Logique de la classe Event  
│   └── user.rb        # Logique de la classe User  
├── app.rb             # Fichier d'exécution principal  
├── README.md          # Documentation  
├── Gemfile            # Gestion des dépendances  
└── Gemfile.lock       # Verrouillage des versions  

## 🚀 Installation & Utilisation
* Installe les dépendances : bundle install
* Lance l'application : ruby app.rb
