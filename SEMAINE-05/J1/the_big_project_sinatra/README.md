# Mon premier site web avec Ruby (The Gossip Project)

## 📝 C'est quoi ce projet ?
Aujourd'hui, j'ai franchi une grande étape : je ne fais plus seulement des petits calculs dans mon terminal noir, j'ai créé un **vrai site web** ! 
Le but est simple : une page où on peut raconter des potins sur les copains. On écrit son nom, son secret, et paf, ça s'affiche sur le site.

## 🛠 Ce que j'ai réussi à faire :
- **Une page d'accueil** : Elle liste tous les secrets que les gens ont écrit.
- **Un formulaire** : Une petite page avec des cases à remplir pour ajouter son propre secret.
- **Une base de données "maison"** : Comme je n'ai pas encore de vraie base de données, j'utilise un fichier texte (.csv) qui enregistre tout pour que les secrets ne disparaissent pas quand j'éteins mon ordi.
