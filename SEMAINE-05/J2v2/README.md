# S5J2 - Mes premières bases de données (SQL)

## 📝 C'est quoi ce projet ?
Après avoir utilisé des fichiers texte (CSV) hier, j'ai découvert aujourd'hui le "niveau pro" pour ranger des informations : les **Bases de Données Relationnelles**. J'ai appris à utiliser le langage **SQL** pour créer, chercher et modifier des données dans un "classeur intelligent" appelé SQLite3.

## 🛠 Ce que j'ai réussi à faire :
- **Plans d'architectures** : J'ai dessiné les structures de 5 sites (Pinterest, blog, etc.) pour comprendre comment les données sont reliées.
- **Interrogateur de données** : J'ai écrit des requêtes SQL complexes pour extraire des informations d'une base de données musicale (Chinook).
- **Création réelle** : J'ai créé ma propre base de données pour une application de blog directement depuis mon terminal.
