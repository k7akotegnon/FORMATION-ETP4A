# S5J2 (Le langage SQL)

## 1. La base de données : un classeur intelligent
J'ai compris qu'une base de données est un ensemble de **Tables** (comme des onglets Excel). La grande différence avec le CSV, c'est que ces tables peuvent être reliées entre elles : c'est pour ça qu'on dit "Relationnelle".

## 2. Le petit lexique que j'ai retenu :
- **Primary Key** : C'est la plaque d'immatriculation unique d'une ligne. Elle permet de ne jamais confondre deux utilisateurs.
- **Foreign Key** : C'est le lien. Ça permet de dire qu'un article appartient à tel utilisateur en notant son numéro d'ID.
- **CRUD** : Les 4 pouvoirs de base sur les données : Créer (INSERT), Lire (SELECT), Modifier (UPDATE), et Supprimer (DELETE).
- **Jointure (JOIN)** : C'est quand on "colle" deux tables ensemble pour voir, par exemple, le nom d'un artiste à côté du titre de son album.

## 3. Mes découvertes techniques :
J'ai appris à utiliser l'extension **SQLite de VSCode** pour tester mes commandes SQL sans passer par le terminal noir, ce qui est beaucoup plus visuel pour un débutant !
