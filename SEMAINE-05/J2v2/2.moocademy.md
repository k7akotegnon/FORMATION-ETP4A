## 2. MOOCademy

*Un système de cours en ligne avec des leçons.*

### Schéma Logique
- **courses** : id, title, description
- **lessons** : id, title, body, course_id (FK vers courses)

*FK = Foreign key*

**Relations :**
- 1-N : un cours a plusieurs leçons

### Requêtes de création (SQL)
```sql
-- Table des cours principaux
CREATE TABLE courses (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  title TEXT,
  description TEXT
);

-- Table des leçons rattachées à un cours (Relation 1-N)
CREATE TABLE lessons (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  title TEXT,
  body TEXT,
  course_id INTEGER,
  FOREIGN KEY (course_id) REFERENCES courses(id)
);
