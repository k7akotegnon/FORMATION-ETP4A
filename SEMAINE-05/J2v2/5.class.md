## 5. The Class

*Gestion simple des inscriptions aux cours.*

## Schéma Logique
- **courses** : id, title
- **students** : id, name, course_id (FK vers courses)

*FK = Foreign key*

**Relations :**
- 1-N : un cours peut avoir plusieurs élèves
- 1-1 : un élève est inscrit à un seul cours

## Requêtes de création (SQL)
```sql
CREATE TABLE courses (id INTEGER PRIMARY KEY AUTOINCREMENT, title TEXT);

-- Table des élèves inscrits à un SEUL cours (Relation 1-N)
CREATE TABLE students (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  name TEXT,
  course_id INTEGER,
  FOREIGN KEY (course_id) REFERENCES courses(id)
);
