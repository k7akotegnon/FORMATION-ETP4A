## 4. The News (Hacker News Style)
*Utilisateurs, liens et commentaires hiérarchiques.*

### Schéma Logique
- **users** : id, name
- **links** : id, url, user_id (FK vers users)
- **comments** : id, content, user_id, link_id, parent_comment_id (FK pour la hiérarchie)

*FK = Foreign key*

**Relations :**
- 1-N : un user peut poster plusieurs links
- 1-N : un link peut avoir plusieurs commentaires
- auto-relation : un commentaire peut avoir un parent_comment_id (commentaire de commentaire)

### Requêtes de création (SQL)
```sql
CREATE TABLE users (id INTEGER PRIMARY KEY AUTOINCREMENT, name TEXT);

-- Table des liens partagés
CREATE TABLE links (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  url TEXT,
  user_id INTEGER,
  FOREIGN KEY (user_id) REFERENCES users(id)
);

-- Table des commentaires avec lien parent pour les réponses (Hiérarchie)
CREATE TABLE comments (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  content TEXT,
  user_id INTEGER,
  link_id INTEGER,
  parent_comment_id INTEGER, -- Permet de répondre à un commentaire précis
  FOREIGN KEY (user_id) REFERENCES users(id),
  FOREIGN KEY (link_id) REFERENCES links(id),
  FOREIGN KEY (parent_comment_id) REFERENCES comments(id)
);
